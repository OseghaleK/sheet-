1. Discrete Probability

Q: Find expected value?
E(X) = Σ[x · P(x)]
Q: Find variance?
Var(X) = E(X²) - [E(X)]²
Store Visits: P(0)=0.1, P(1)=0.4, P(2)=0.4, P(3)=0.1
E(X) = 0×0.1 + 1×0.4 + 2×0.4 + 3×0.1 = 1.5
E(X²) = 0×0.1 + 1×0.4 + 4×0.4 + 9×0.1 = 2.9
Var(X) = 2.9 - (1.5)² = 0.65 → σ = √0.65 = 0.806

2. Binomial Problems

When to use: Fixed trials, two outcomes, constant p
Q: Exactly k successes?
P(X=k) = C(n,k) × pk × (1-p)n-k
Q: At least one success?
P(X≥1) = 1 - (1-p)n
Rifleman: n=4, p=0.8
P(2 hits) = C(4,2)×0.8²×0.2² = 6×0.64×0.04 = 0.1536
μ = 4×0.8 = 3.2, σ² = 4×0.8×0.2 = 0.64

3. Normal Distribution

Q: Find probability?
Z = (X - μ)/σ → Use Z-table
Q: Find X from percentile?
X = μ + Z × σ
Test Scores: N(75, 10²)
P(X > 85)? Z = (85-75)/10 = 1 → P = 0.1587
Top 5% cutoff? Z = 1.645 → X = 75 + 1.645×10 = 91.45

4. Central Limit Theorem

Applies when: n ≥ 30 or population normal
Q: Sample mean probability?
Z = (X̄ - μ)/(σ/√n)
Bottles: μ=500ml, σ=10ml, n=25
P(X̄ < 497)? Z = (497-500)/(10/5) = -1.5 → P = 0.0668
Q: Find sample size?
n = (Z × σ / margin of error)²

5. Confidence Intervals

Q: Population mean CI?
X̄ ± Z × (σ/√n)
Sample: n=100, X̄=50, σ=5, 95% CI
50 ± 1.96×(5/10) = 50 ± 0.98 → (49.02, 50.98)
