1. Probability Distributions

Expected Value (Mean): E(X) = Σ[x · P(x)]
Example: Store visits: P(0)=0.1, P(1)=0.4, P(2)=0.4, P(3)=0.1
E(X) = 0×0.1 + 1×0.4 + 2×0.4 + 3×0.1 = 1.5
Variance: Var(X) = E(X²) - [E(X)]²
Steps:
  1. Find E(X²) = Σ[x² · P(x)]
  2. Calculate Var(X) = E(X²) - [E(X)]²
  3. Standard Deviation: σ = √Var(X)
E(X²) = 0²×0.1 + 1²×0.4 + 2²×0.4 + 3²×0.1 = 2.9
Var(X) = 2.9 - (1.5)² = 0.65, σ = √0.65 = 0.806

2. Binomial Distribution

Use when: Fixed trials, two outcomes, constant probability
P(X = k) = C(n,k) × p^k × (1-p)^(n-k)
Rifleman Example: n=4, p=0.8, find P(exactly 2 hits)
P(X = 2) = C(4,2) × (0.8)² × (0.2)² = 6 × 0.64 × 0.04 = 0.1536
Mean: μ = n × p
Variance: σ² = n × p × (1-p)
Die Example: n=54, p=1/6
μ = 9, σ² = 7.5, σ = 2.739
At least one success: P(X ≥ 1) = 1 - P(X = 0) = 1 - (1-p)^n

3. Normal Distribution

Z-Score: Z = (X - μ) / σ
Example: X ~ N(10, 2²), find P(11 < X < 13)
Z₁ = (11-10)/2 = 0.5, Z₂ = (13-10)/2 = 1.5
P(11 < X < 13) = P(Z < 1.5) - P(Z < 0.5) = 0.9332 - 0.6915 = 0.2417
Finding X from percentile:
X = μ + Z × σ
Bottom 1.5%, μ=400, σ=100
Z = -2.17, X = 400 + (-2.17)(100) = 183
Finding μ when σ known:
μ = X - Z × σ
2.5% below 85, σ=6
Z = -1.96, μ = 85 - (-1.96)(6) = 96.76

4. Central Limit Theorem (CLT)

Sample means are normally distributed regardless of population shape (n ≥ 30)
Standard Error: σx̄ = σ / √n
Z-score for sample mean: Z = (X̄ - μ) / (σ/√n)
Actuarial Salaries: μ=$40,000, σ=$5,000
• Individual P(X > $45,000): Z = (45000-40000)/5000 = 1 → P = 0.1587
• Group of 9 P(X̄ > $45,000): Z = (45000-40000)/(5000/√9) = 3 → P = 0.0013

5. Expected Value Applications

Lottery Pricing: Price = (Total Prizes + Desired Profit) / Number of Tickets
Example: 1000 tickets, $1500 prizes, want $2500 profit
Price = (1500 + 2500) / 1000 = $4.00 per ticket
